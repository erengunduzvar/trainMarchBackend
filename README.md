# API TABLE
## Users

| HTTP | Endpoint | Description         |
| ----------- | -------- | ------------------- |
| GET         | /users| Get all users|
| GET         | /users:userId     | Get one user|
| POST        | /users        | Create user|
| PUT         | /users:userId     | Update user|
| DELETE      | /users:userId     | Delete user|
